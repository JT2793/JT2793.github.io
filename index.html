All in one Cheatsheet for HTML & CSS

HTML Tags

Tag Description
<html> All html code falls within this tag
<head> Where the following live...
<nav> Navigation of HTML page falls within
<title> Name of project (will appear on browser tab)
<link> connect resources outside the current document to this one
<body> Where the Main content of your webpage lives
<header main footer> foundations for HTML structure (semantic)
<h1 - h6> headings (decending size)
<p> paragraphs/text
<b> bold
<i> italic
<u underline
<br> break to next line
<span> 'inline' container
<div> block/box container
<a href=""> Anchor / links
ul, ol, li, Un/ordered lists & list items
To Embed
<img src="" alt=""> embed an image. Doesn't reuire closing tag
<video src="" controls> embed a video
width="" height="" Should be used in conjunction with img/vid
Tables
<table> the wrapper element for HTML tables
<tr> Table rows
<th> Table header
<td> Table data
<tbody & tfoot> used to split table data (semantics, readability)
